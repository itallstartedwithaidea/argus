**VirusTotal для фейковых профилей и постов в социальных сетях — бесплатно, с открытым кодом, нативно на Cloudflare**

📊 **Отчёты**: [googleadsagent.ai/tools/argus/reports](https://googleadsagent.ai/tools/argus/reports)

## Что он делает

ARGUS анализирует любой профиль, пост или паттерн взаимодействия в социальных сетях — LinkedIn, Reddit, Instagram, X, Facebook, YouTube, Quora, TikTok и Pinterest. Возвращает **взвешенный балл доверия (0–100)** с полным обоснованием, разбивкой по движкам детекции, анализом комментаторов и объяснением штрафов.

Отмеченный контент получает публичную страницу, индексируемую Google, с SEO-дружественными URL. Весь анализ представлен как алгоритмическое мнение — не вердикт. Владельцы профилей могут оспорить в любое время.

**Каждая опубликованная страница проверена и одобрена человеком.**

---

## Ключевые функции

- **4 движка детекции**: Изображение, Текст, Поведенческий, Сетевой — каждый оценивается независимо
- **Анализ комментаторов**: Каждый видимый комментатор анализируется на паттерны ботов, координацию, подлинность
- **Алгоритм взвешенной оценки**: `Сетевой(35%) + Поведенческий(30%) + Изображение(20%) + Текст(15%) - Штрафы`
- **Система штрафов**: Автоматические вычеты за непроверяемые данные, паттерны ботов, фейковое взаимодействие и т.д.
- **Режим приватного профиля**: Вставьте контент + скриншоты для профилей, которые ARGUS не может просканировать
- **Анализ Claude AI**: На базе Anthropic Claude для глубокого анализа контента
- **Динамические публичные отчёты**: Wiki-страницы, генерируемые на лету через Cloudflare Workers
- **SEO-дружественные URL**: `/report/reddit-landing-page-navigation-conversion-85`
- **Панель администратора**: Защита OAuth с аналитикой Chart.js, группировкой, разрешением споров
- **Загрузка доказательств**: Хранение R2 для файлов доказательств при оспаривании
- **Email-уведомления**: Подробные результаты анализа отправляются через Resend
- **Динамическая карта сайта**: Автоматически генерируется из опубликованных отчётов
- **Расширение для браузера**: Расширение Chrome для быстрого анализа

## Алгоритм оценки

В отличие от общего ИИ-скоринга, ARGUS использует **детерминированный взвешенный композит**:

```
Final Score = Weighted Composite - Penalties

Weighted Composite:
  Network Engine  × 0.35
  Behavioral      × 0.30
  Image           × 0.20
  Text            × 0.15

Penalties (applied automatically):
  Account age unverifiable:      -8
  No visible posting history:    -10
  Private submission:            -15
  Page couldn't be fetched:      -12
  Very limited content:          -10
  Bot activity detected:         -up to 20
  AI-generated content:          -up to 15
  Fake engagement detected:      -up to 20
  >50% suspicious comments:      -15
  High followers + few posts:    -18
  New account + high engagement: -20
```

## Движки детекции

| Движок | Анализирует | Ключевые сигналы |
|--------|----------|-------------|
| **Изображение** | Фото профиля, изображения постов | Артефакты GAN, стоковые фото, провенанс C2PA, аномалии EXIF |
| **Текст** | Био, посты, комментарии | Текст, сгенерированный ИИ, стилометрическая согласованность, шаблонный язык |
| **Поведение** | Паттерны активности | Возраст аккаунта vs активность, частота публикаций, коэффициенты взаимодействия |
| **Сеть** | Связи, взаимодействия | Соотношения подписчиков, индикаторы координации, сигнатуры бот-сетей |

## Юридическая архитектура

Каждая страница защищена:

1. **«Алгоритмический анализ, не вердикт»** — отображается на каждой странице
2. **Полное раскрытие методологии** — открытый исходный код, публично проверяемый
3. **Прозрачная оценка** — взвешенный композит с отображёнными штрафами
4. **Заметный механизм оспаривания** — на видном месте, не спрятан
5. **Редакторская проверка человеком** — администратор одобряет каждую опубликованную страницу

## Процесс оспаривания / отказа

Любой может подать оспаривание на `/tools/argus/dispute` с загрузкой файлов (R2).

| Уровень | Доказательства | Результат |
|------|----------|---------|
| **Уровень 1** | Государственное удостоверение + верификация платформы | Автоматическое одобрение удаления |
| **Уровень 2** | Присутствие на нескольких платформах, подтверждение работодателя | Проверка человеком |
| **Уровень 3** | «Я настоящий» без доказательств | Оспаривание отклонено |
